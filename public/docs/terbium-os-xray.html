1. Module Overview

Terbium OS is Skolyn's X-ray and digital radiography analysis module, engineered for the highest-volume imaging modality in healthcare. Named after Terbium (Tb, atomic number 65), an element prized for its luminescence in imaging phosphors, the module reflects its critical role in projection radiography analysis. X-ray is the most commonly performed medical imaging examination worldwide, accounting for approximately 60% of all imaging studies, making efficient and accurate AI analysis at scale essential. Terbium OS supports 10 anatomical subtypes covering chest, musculoskeletal, mammographic, dental panoramic, bone age, abdominal, cervical spine, thoracic/lumbar spine, pelvic, and pediatric applications. The module processes single-projection and multi-view studies with sub-5-second turnaround time per image.

3. Architecture

3.1 Multi-Scale Encoder

The defining architectural feature of Terbium OS is the Multi-Scale Encoder (MSE), designed to handle the unique challenge of projection radiography: clinically significant findings span multiple spatial scales within a single 2D image. A pneumothorax may occupy 40% of the hemithorax, while a subtle hairline fracture is only a few pixels wide. The MSE processes each radiograph through four parallel resolution branches (full resolution, 1/2, 1/4, 1/8) with shared backbone weights (EfficientNet-V2-L), feature pyramid network (FPN) connections for cross-scale information exchange, and a final multi-scale attention aggregation layer that learns to weight scale-specific features based on finding type. The MSE achieves state-of-the-art performance on CheXpert, MIMIC-CXR, and VinDr-CXR benchmarks while maintaining sub-3-second inference time on a single A30 GPU.

3.2 Projection-Aware Analysis

X-ray images are 2D projections of 3D anatomy, which creates unique challenges: overlapping structures, magnification effects, and projection-dependent appearance of pathologies. Terbium OS incorporates projection-aware modules that adjust analysis parameters based on the view position (PA vs AP, lateral, oblique, cross-table lateral). For multi-view studies (e.g., PA + lateral chest, or multiple mammographic views), the Multi-View Fusion module jointly analyzes all projections, resolving ambiguities that are present in single-view analysis. For example, a retrocardiac opacity visible on the lateral view provides confirmatory evidence for a subtle finding on the PA view.

5. Chest X-Ray Deep-Dive

CXR is the most impactful subtype in Terbium OS, processing millions of studies annually across Skolyn deployments. The CXR engine detects and localizes 14 distinct pathologies simultaneously, generating bounding box annotations with confidence scores for each finding. The engine also performs automated measurements including cardiothoracic ratio (CTR) for cardiomegaly quantification, tracheal midline deviation for tension pneumothorax assessment, and tube/line tip position verification (ETT, NG tube, central venous catheter, chest tube). For follow-up studies, temporal comparison automatically aligns the current and prior CXR using affine registration and highlights interval changes in a difference map overlay.

6. Mammography Screening

Terbium OS's mammography subtype supports both screening (2D digital mammography) and diagnostic workflows. For screening mammography, the module processes all four standard views (RCC, LCC, RMLO, LMLO) jointly, providing per-view and per-breast BI-RADS classifications with overall assessment. The module performs automated breast density classification (ACR A-D) using a dedicated DenseNet model, mass detection with morphological characterization (shape, margin, density), calcification detection with distribution pattern analysis (clustered, regional, segmental, diffuse), and architectural distortion identification. All findings include CAD marks on the mammographic image for radiologist review.
