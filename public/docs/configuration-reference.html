1. Overview

This Configuration Reference provides a comprehensive listing of all configurable parameters available in the Skolyn platform. Parameters are organized by functional area and can be set through the Management Console UI, the Skolyn CLI, environment variables, or the Helm values file (for Kubernetes deployments). Each parameter includes its type, default value, valid range, and a description of its behavior. Configuration changes are version-controlled and audited — all modifications create an immutable audit trail entry with the before/after state, the identity of the operator, and a timestamp. Critical configuration changes (those affecting security, authentication, or data handling) require a secondary approval workflow before taking effect in production.

Configuration Precedence: Helm values < Environment variables < ConfigMap/Secret < Management Console (runtime). Runtime changes via the console take highest priority and persist across pod restarts via the configuration database.

3. Inference Engine Configuration

ParameterTypeDefaultDescription
inference.batch_sizeInteger4Number of studies processed concurrently per GPU. Increase for smaller studies, decrease for multi-sequence MRI.
inference.timeoutInteger300Maximum inference time per study in seconds. Studies exceeding this timeout are marked as failed.
inference.precisionStringfp16Inference precision mode: fp32 (full), fp16 (half), int8 (quantized). fp16 recommended for optimal speed/accuracy balance.
inference.gpu_memory_fractionFloat0.85Maximum fraction of GPU memory allocated per worker (0.0-1.0). Reserve 15% for system overhead.
inference.queue.max_depthInteger1000Maximum pending studies in the processing queue before rejecting new submissions.
inference.queue.priority.statInteger0Priority weight for STAT studies (0 = highest priority). STAT studies preempt normal queue.
inference.explainability.enabledBooleantrueGenerate attention map (Grad-CAM++) overlays for findings. Adds ~5% processing time.
inference.explainability.resolutionStringhighAttention map resolution: low (64x64), medium (128x128), high (256x256).
